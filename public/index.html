Software Linup

  • Atom : switching from Emacs, still using emacs for a few minor modes (mainly GNU-Octave).
  • TeXmaker : mainly because it’s cross platform and provides super easy access to multiple compile options, shortcuts etc. But I hope a nice Atom package shows up.
  • Freeplane : simple and actively developped.
  • GNU-Octave : who needs Matlab ? Still mostly compatible with it.
  • CitySim : Has the very big advantage of being integrable into a multi-software workflow (Rhino, autoCAD for preprocessing, Octave for post-process etc.).
  • Rhino : my CAD software of choice for designing urban blocks. No good OS alternative as far as I know.
